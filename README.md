# Personal Portfolio Website

## Overview

This is a modern and responsive personal portfolio website developed to showcase my academic background, technical skills, projects, and professional information. The website serves as an online portfolio where visitors can learn more about me, explore my work, and connect with me through various contact channels.

The portfolio features a clean user interface, smooth animations, responsive design, and a professional layout suitable for students, developers, and designers.

## Features

* Responsive Design for desktop and mobile devices
* Professional Hero Section with profile image
* About Me section
* Technical and Professional Skills
* Educational Background Timeline
* Featured Projects Showcase
* Contact Information Section
* GitHub and LinkedIn Integration
* Smooth Scrolling Navigation
* Modern UI with animations and visual effects

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts
* Responsive Web Design

## Sections Included

### Home

Introduces the portfolio owner with a professional profile image and personal introduction.

### About

Provides personal background, educational information, and career objectives.

### Skills

Displays technical skills, design tools, programming languages, and soft skills.

### Education

Shows academic qualifications and educational timeline.

### Projects

Highlights completed projects with descriptions and technologies used.

### Contact

Provides email, phone number, location, GitHub profile, and LinkedIn profile.
